# Правила размещения: HEADER-1xN (THT)

## Соответствие имён корпуса

Считать этим типом корпуса всё, что пользователь обозначает как:
- HEADER-1xN, PINHEADER-1xN, “гребёнка 1xN”, “штырьки 1 ряд”

## Footprint

- N выводов в одну линию
- Шаг: 1 отверстие между соседними выводами
- N определяется пользователем из списка pins

## Допустимая ориентация

- По X или по Y

## Корпус и перекрытие отверстий

По умолчанию:
- bodyKeepout: линия отверстий непосредственно “за” разъёмом со стороны пластика, шириной 1 отверстие и длиной N

Если пользователь сообщает, что это “угловой” разъём (90°), увеличить bodyKeepout до 2 отверстий в глубину со стороны выхода проводов.

## Рекомендации

- Если разъём — интерфейс наружу, стараться ставить его на краю минимального bbox
- Пины с питанием (VCC/GND) группировать с краю, если это не мешает трассировке
