# DIYLayout Project - Gemini CLI Standards

Этот проект предназначен для автоматизированного проектирования раскладок электронных компонентов на макетных платах (perfboards). 

## Правила

1. Всегда используй инструменты из папки .gemini/skills/autorouter/tools/ для автоматизации работы.
2. Всегда используй инструменты из папки .gemini/skills/perfoboard-layout-optimizer/tools/ для автоматизации работы.
3. Всегда используй инструменты из папки .gemini/skills/perfoboard-layout-optimizer/viewer/ для визуализации работы.
4. Всегда используй инструменты из папки .gemini/skills/perfoboard-layout-optimizer/tools/ для автоматизации работы.
5. Всегда используй инструменты из папки .gemini/skills/perfoboard-layout-optimizer/tools/ для автоматизации работы.
6. Всегда используй агент schematic-interpreter для формирования списка деталей и соединений
7. Всегда используй агент zone-segment-planner для разбивки всей площади платы на зоны и этажи внутри зон
8. Всегда используй агент rule-to-json-converter для конвертации правил из .md в .json

## Алгоритм

1. Передай управление агенту schematic-interpreter для формирования списка деталей и соединений
2. Передай управление агенту rule-to-json-converter для конвертации правил из .md в .json
3. Передай управление агенту zone-segment-planner для разбивки всей площади платы на зоны и этажи внутри зон
4. Воспользуйся скилом segment-placement-engine для определения координат зон и этажей на плате
5. Воспользуйся скилом component-placement-engine для размещения компонентов внутри сегмента
6. Воспользуйся скилом pin-escape-generator для вывода дорожек из плотной зоны пинов
7. Воспользуйся скилом autorouter для соединения всех компонентов
