# Правила размещения: TO-220 (THT)

## Соответствие имён корпуса

Считать этим типом корпуса всё, что пользователь обозначает как:
- TO-220, TO-220-3, TO-220-5, “силовой транзистор/регулятор TO-220”

Если число выводов не 3, брать фактическое количество из списка `pins`.

## Footprint

- Выводы в одной линии
- Шаг между соседними выводами: 1 отверстие (2.54 мм)
- Количество выводов по умолчанию: 3
- Если пользователь укажет иное (TO-220-5 и т. п.), количество берётся из `pins`

## Допустимая ориентация

- Линия выводов по X или по Y
- Корпус ориентируется так, чтобы “плечо/радиаторная площадка” находилась за линией выводов

## Корпус и перекрытие отверстий (keepout)

По умолчанию:
- bodyKeepout отсутствует; заняты только отверстия трёх выводов
- Корпус не перекрывает дополнительные отверстия

Если пользователь явно запросит радиатор или увеличенный тепловой зазор:
- добавить прямоугольную keepout-зону за линией выводов (глубина 2–4 отверстия)
- при необходимости располагать корпус ближе к краю минимального bbox

## Монтажное отверстие корпуса (optional)

По умолчанию монтажное отверстие не используется.
Если пользователь хочет фиксировать корпус винтом:
- зарезервировать дополнительное отверстие под винт (указать координату в результате)
- не проводить дорожки непосредственно под “юбкой” крепления

## Изгиб выводов

По умолчанию любые смещения выводов запрещены.
Если пользователь явно разрешит:
- смещение крайних выводов на 1 отверстие перпендикулярно линии для разведения (dogleg)

Ограничения по умолчанию:
- центральный вывод остаётся на исходной линии
- не допускать пересечения выводов между собой

## Тепловые и электрические рекомендации

- Высокотепловые элементы (линейные стабилизаторы, силовые транзисторы) располагать ближе к краю, для лучшего отвода тепла и доступа воздуха
- Дорожки с большим током делать максимально короткими и толстими (минимизировать “змейки”)
- Не помещать чувствительные аналоговые узлы (ОУ входы, датчики низкоуровневые) вплотную к корпусу TO-220
- Если есть GND-шина, отдавать приоритет прямым и коротким соединениям корпуса/затвора/источника по требованию схемы

## Полярность/порядок выводов

- Имена выводов берутся из `pins` (например для регуляторов: IN/GND/OUT; для MOSFET: G/D/S)
- Сохранять маркировку в результатах, соответствуя фактической нумерации и ориентации корпуса

## Рекомендации для оптимизации площади

- При необходимости теплоотвода располагать TO-220 ближе к краю будущего bbox
- Соседние элементы, связанные с TO-220 (датчик тока, делители, драйверы) располагать вокруг линии выводов, чтобы сократить трассы
